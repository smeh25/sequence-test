# sequence-test
This program generates a sequence of n random numbers and prompts the user to input the next number in the sequence.

  What's included:

  18 distinct sequence types, randomly selected each round:
  - Arithmetic (ascending & descending)
  - Geometric ×2, ×3, ×4
  - Perfect squares & cubes
  - Fibonacci-like (custom starting pair)
  - Triangular numbers
  - Powers of 2 & 3
  - Prime numbers
  - Alternating steps (+d1, +d2, +d1, …)
  - Odd-number differences (2nd differences constant)
  - Double + constant, Triple − constant
  - Quadratic (an² + bn + c)
  - Squares × coefficient + offset
